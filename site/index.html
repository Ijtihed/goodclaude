goodclaude

positive reinforcement for claude code.

Install + run

npm install -g goodclaude
copied 
goodclaude
copied

How it works

  • 01

    Press F8 to instantly reward Claude. Pastes a message, plays a chime, confetti.

  • 02

    Or click the tray icon to spawn a golden star. Drag it to your terminal, click to drop.

  • 03

    Edit ~/.goodclauderc to change hotkeys or messages.

Messages

# 🌟 Great job Claude! Keep it up!

# 🍪 Here's a cookie for being awesome

# ⭐ You're doing amazing work

# 🏆 Claude MVP of the day

# 💪 That was clean code, well done
